# Minimum Spanning Tree

## Overview

This project implements the **Minimum Spanning Tree (MST)** algorithm to find the minimum-cost set of edges required to connect all vertices in a weighted graph without creating cycles.

## Features

* Efficient graph traversal and edge selection
* Minimum cost network construction
* Cycle-free spanning tree generation

## Applications

* Network design and optimization
* Road and railway planning
* Telecommunications

## Time Complexity

* Kruskal's Algorithm: **O(E log E)**
* Prim's Algorithm: **O(E log V)**

Where:

* **V** = Number of vertices
* **E** = Number of edges
